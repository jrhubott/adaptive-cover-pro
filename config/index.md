## What is Adaptive Cover Pro?

Adaptive Cover Pro automates window coverings based on:
- **Solar position** (sun tracking)
- **Weather conditions** (wind, rain, cloud)
- **Occupancy** (motion sensors)
- **Time of day** (adaptive time window)
- **Manual override** (user control with auto-reset)

It replaces basic timers with intelligent sun-aware automation.
