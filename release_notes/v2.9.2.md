## 🎯 Suppress Glare Control in Low Light (Issue #65)

### ✨ Features

- **Suppress glare control in low light** — New optional toggle in Climate Settings. When enabled, uses your configured weather entity, lux sensor, or irradiance sensor to detect when there is no real direct sun. Covers return to their default position instead of closing to block non-existent glare (e.g., on rainy days or in dark rooms).
  - Works independently of climate mode — useful without enabling full temperature-based climate control
  - Requires at least one of: weather entity, lux sensor, or irradiance sensor to be configured

### 🐛 Bug Fixes

- **Config flow menu** — Blind spot, interpolation, and weather options now appear correctly in the options menu before Save & Exit
- **Translations** — Renamed "Automation" climate menu option to "Climate Configuration" for clarity

### 📚 Documentation

- Improved documentation for transparent blinds, field of view, and vertical blind diagram (Issue #87)
