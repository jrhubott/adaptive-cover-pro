## 🪟 Window Sill Height Support

### ✨ Features

- **New `sill_height` parameter** for vertical blinds — accounts for windows that don't start at floor level
  - Set to the height of your window sill above the floor (0.0–3.0m, default 0.0)
  - When configured, the blind opens higher than it would otherwise, as the sill already blocks low-angle sunlight from reaching that part of the floor
  - Fully backward compatible — existing installations are unaffected (default 0.0 = floor-level window)
  - Not applicable to horizontal awnings or tilt covers

### 🔧 How It Works

A window sill at height S means sunlight entering near the bottom of the glass cannot reach closer than `S / tan(sun elevation)` meters from the wall. The integration accounts for this "free" protection and adjusts the blind position accordingly.

**Example:** A window with its sill 1 meter above the floor at 45° sun elevation gets 1 meter of natural sun blocking from the sill — so the blind can open 1 meter more than it would for a floor-level window with the same glare target.
