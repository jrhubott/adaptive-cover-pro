# v2.9.1 — Config Flow Polish

Small fixes to the config flow introduced in v2.9.0.

## 🐛 Bug Fixes

- **Device Association** was missing from the options menu — it now appears between Cover Entities and Cover Geometry
- **Automation** menu item renamed to **Schedule & Timing** for clarity

## ✨ Improvements

- Each options screen now returns to the main menu after submitting, so you can edit multiple sections in one session
- Added **Save & Exit** to the options menu to save all changes and close when you're done
