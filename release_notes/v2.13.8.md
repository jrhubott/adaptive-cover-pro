# v2.13.8 — Configuration Summary & UI Improvements

## 🎯 Summary

Improves user experience with better configuration visibility and reorganized settings screens. Configuration summary now shows the complete day/night position cycle, and position-related settings are consolidated on the Position Settings screen.

## ✨ Features

- **Configuration summary now displays complete position cycle**:
  - Shows when automatic sun tracking resumes after sunrise (with time offset if configured)
  - Annotates sunset and sunrise transitions with their offsets
  - Example: "After sunset (+30 min) → 80%" and "After sunrise (+60 min) → 0% (tracking resumes)"
  - Helps users understand position behavior timing without digging into individual settings

## 🔧 UI/UX Improvements

- **Position Settings screen reorganization**:
  - Moved "Open/Close threshold" from Schedule & Timing screen to Position Settings screen
  - Now positioned logically after "Sunrise Offset" with other position-related thresholds
  - Rationale: This setting controls position behavior (when to trigger open vs. close), not scheduling
  - Makes position-related controls consistent on one screen

## 🐛 Bug Fixes

- **False manual override on automation positioning** (from v2.13.8-beta.3):
  - Fixed race condition where automation's own position commands were detected as manual overrides
  - Manual override indicator no longer activates during normal automated positioning
  - Motor rounding (1-3%), coarse position granularity (5% steps), and position reporting imprecision now properly tolerated
