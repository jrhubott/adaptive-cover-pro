## 🐛 Bug Fix: Position Limit Sliders Default Correctly

### 🐛 Bug Fixes
- **Maximum Position slider now defaults to 100%** — Previously, when configuring position limits for the first time, the Maximum Position slider would start at 1% (the slider minimum) instead of 100%. The slider now defaults to 100%, which is the no-op value meaning "no upper limit".
- **Minimum Position slider now defaults to 0%** — Similarly, the Minimum Position slider now defaults to 0% (no lower limit) instead of showing no value.
