## Manual Override on Somfy My Stop

### ✨ Features

- **Manual override via stop_cover for Somfy RTS covers (#202):** Pressing STOP on a non-position-capable cover (e.g. Somfy RTS) now correctly triggers a manual override. Previously, Somfy RTS covers move to the hardware "My" preset when STOP is pressed, but never report a new position — so the integration had no way to detect the user's intent and would fight the preset on the next solar calculation. ACP now listens for `cover.stop_cover` service calls on tracked covers and, when a configured "My position" value is set, immediately flags the cover as manually overridden at that position. Integration-originated stop calls (e.g. from ACP's own safety handlers) are filtered out so they don't falsely trigger an override.

### 🧪 Testing

- 360+ line test suite covering stop_cover detection, ACP-originated call filtering, manual override flagging, and reconciliation target alignment (`tests/test_my_position.py`)
