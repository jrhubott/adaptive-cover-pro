## 🐛 Bug Fixes

### ✨ Improvements

- **Motion control now accepts occupancy sensors** (#52) — mmWave and other occupancy sensors (declared as `occupancy` device class rather than `motion`) can now be selected in the motion control sensor list. Previously only sensors with the `motion` device class were accepted.

### 🐛 Bug Fixes

- **Sunset position can now be left blank** (#55) — Clearing the sunset position field to use the default cover position no longer shows a validation error. The field correctly accepts an empty value.
