## 🧹 Diagnostic Sensor Cleanup

### 🔧 Improvements

- Removed redundant attributes from the `sun_elevation` diagnostic sensor — `valid_elevation`, `blind_spot_elevation`, and `in_blind_spot` were duplicating data already available on the dedicated `sun_validity` sensor. The `sun_elevation` sensor now only exposes `min_elevation` and `max_elevation` (when configured), keeping each sensor focused on its own domain.

### 📚 Documentation

- Updated README diagnostic sensor table to reflect the simplified `sun_elevation` attribute set and direct users to `sun_validity` for elevation validity and blind spot status.

### 🧪 Testing

- All 313 tests passing
- Attribute-only change — no logic modifications
