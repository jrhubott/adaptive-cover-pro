# v2.13.8-beta.3 — False Manual Override Fix

## 🎯 Summary

Fixes a critical race condition where the automation's own position commands were being misidentified as manual overrides. This was causing the manual override indicator to turn on continuously even during normal automated positioning.

## 🐛 Bug Fixes

- **False manual override on automation positioning**: Covers now settle to their commanded position without triggering a false manual override detection. Motor rounding (1-3%), coarse position granularity (5% steps), and position reporting imprecision are properly tolerated.

  **Root cause**: When a cover reached its commanded position within the position tolerance (±3%), the system cleared the "waiting for target" flag but immediately ran the override check on the same state-change event with the flag already cleared. A 2% difference between commanded (72%) and actual (70%) position would trigger override detection even though the cover was responding correctly to the automation's command.

  **Fix**: Two complementary mechanisms:

  1. **`_target_just_reached` guard**: When the cover reaches its target, skip the manual override check for that specific state-change event.
  2. **Tolerance floor**: The manual override threshold is now at least 3% (matching the position tolerance), so small motor rounding never triggers false positives independently.

## 📝 Notes for Testers

This beta focuses on eliminating false positive manual override triggering. Please test:

1. **Normal automation positioning**: Covers should move smoothly without triggering manual override
2. **Coarse-granularity covers** (5%, 10% steps): Should not trigger overrides when settling to nearest step
3. **Genuine manual changes**: Manually moving covers by >5% should still be detected as manual overrides
4. **Rapid sun position changes**: Multiple quick positioning commands should not cascade false overrides

If you experience any issues, please open a GitHub issue with:

- Log excerpts showing override detection
- Cover positioning details (min/max position, delta settings)
- Home Assistant version
