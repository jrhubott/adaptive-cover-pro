## 🎯 HA 2026.5 Compatibility Fix

### 🐛 Bug Fixes

- **Integration fails to load on Home Assistant 2026.5+ (#307):** `state_attr` was
  removed from the public Python API (`homeassistant.helpers.template`) in HA 2026.5,
  causing top-level import failures in the coordinator, cover command manager,
  cover provider, and climate provider — making the entire integration non-functional
  after upgrading to 2026.5. A local `state_attr` helper is now defined in `helpers.py`
  with an identical contract and all four call sites route through it. Import smoke
  tests and a forbidden-import guard are added to catch any regression in CI.

### 🧹 Chores

- Add HA beta channel to CI test matrix to catch upstream API removals earlier.
- Diagnostics file is now required on bug reports; a workflow auto-labels and
  auto-closes issues missing an attachment after 7 days.
