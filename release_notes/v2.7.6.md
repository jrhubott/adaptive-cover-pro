## 🎯 ControlMethod Enum — Expanded Cover Position Tracking

### ✨ What's New

**The `control_method` sensor now accurately reflects what is driving your cover position at all times**, not just climate mode strategy.

Previously the sensor only reported `intermediate`, `summer`, or `winter` (and `intermediate` never updated when climate mode was off). It now reports one of seven values in priority order:

| Value             | Meaning                                                                           |
| ----------------- | --------------------------------------------------------------------------------- |
| `force_override`  | A safety binary sensor is active; cover moves to the override position            |
| `motion_timeout`  | No occupancy detected after the configured timeout; cover returns to default      |
| `manual_override` | You manually moved the cover; automatic control is paused                         |
| `summer`          | Climate mode: temperature above max threshold; cover closes to block heat         |
| `winter`          | Climate mode: temperature below min threshold; cover opens for solar heat gain    |
| `solar`           | Sun is within the field of view; cover follows the calculated sun-position        |
| `default`         | Sun is outside FOV, elevation limits, blind spot, or sunrise/sunset offset window |

### 🐛 Bug Fixes

- **Fixed stale control method** — `control_method` previously could remain set to `summer` or `winter` from a previous cycle even after conditions changed. It is now recalculated every update cycle.
- **Renamed `intermediate` → `solar`** — the previous default value was unclear; `solar` accurately describes that the cover is tracking the sun's calculated position.

### 📦 Installation

Download `adaptive_cover_pro.zip` from the release assets and extract to your `custom_components` directory, then restart Home Assistant.

> ⚠️ **Breaking change:** If you have automations or template sensors referencing `intermediate` as a `control_method` value, update them to `solar`.
