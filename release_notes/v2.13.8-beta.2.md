# v2.13.8-beta.2

## 🐛 Bug Fixes

- **Motion Status sensor now shows correct state after reload** — Previously showed `waiting_for_data` when motion sensors were on at reload time. The integration now reads the current state of each configured motion sensor during startup and initializes the motion manager accordingly. When any sensor is on → shows `motion_detected`. When all are off → shows `no_motion`.

- Home Assistant 2024.5.0+
