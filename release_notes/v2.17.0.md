## ✨ My Position Support (Somfy RTS)

### ✨ Features

- **Somfy "My" position support** (#199): RTS motors store a user-programmed
  "My" preset that ACP can now deliberately trigger. Set *My position value* to
  the position you programmed on the remote, then enable *Use My at sunset*
  and/or *Slot N: use My position* on any custom position slot. On non-position-
  capable covers ACP sends `cover.stop_cover` while stationary to trigger My;
  position-capable covers gracefully fall through to `set_cover_position`.

### 🧪 Testing

- Added `tests/test_my_position.py` with 27 tests covering send_my_position,
  _prepare_service_call routing, build_special_positions, CustomPositionHandler
  use_my path, DefaultHandler sunset_use_my path, and regression guards
