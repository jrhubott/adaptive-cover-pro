## 🎯 Geometry Limits, Sun Position Attributes, and Redundant Command Fix

### 🔑 Highlights

- Geometry field limits raised to 50 m and switched to box input, enabling accurate configuration for large buildings (#301 #302).
- `Start Sun` and `End Sun` sensors now expose the sun's azimuth and elevation at the boundary moments of the active tracking window (#305).
- Custom position covers no longer send `set_cover_position` on every sun update — the regression that caused audible relay clicks every few seconds is fixed (#290).

### ✨ Features

- **Raise geometry field limits to 50 m, switch to box input (#301 #302):**
  Window height, window width, sill height, and distance to shaded area were previously capped at 3–6 m with slider input. All four fields are now capped at 50 m and use numeric box input. This supports atypical large spaces — sports halls, atria, and deep industrial environments — where the previous limits made accurate geometry configuration impossible.

- **Expose sun azimuth and elevation on Start Sun and End Sun sensors (#305):**
  The `Start Sun` and `End Sun` timestamp sensors now include `azimuth` and `elevation` state attributes, reflecting the sun's actual position at the moments it enters and exits the active tracking window. These values are clipped by every input `solar_times()` already considers (azimuth FOV, elevation limits, sunrise/sunset offsets), giving the Lovelace card the data it needs to render an accurate FOV wedge without duplicating that logic.

### 🐛 Bug Fixes

- **Custom position covers no longer send redundant commands (#290):**
  Custom position covers were sending `set_cover_position` on every sun update (every few seconds), causing audible relay clicks. The regression landed in v2.18.3 when `CustomPositionHandler` gained `bypass_auto_control=True` — this flag cascaded to `force=True` in the position context, bypassing the same-position short-circuit in `CoverCommandService`. The fix scopes `force=True` to safety handlers only (FORCE_OVERRIDE and WEATHER), restoring the short-circuit for all other handlers including custom positions.
