## 🎯 Auto-prune legacy orphaned entities on upgrade

### 🐛 Bug Fixes
- **Automatic cleanup of duplicate unavailable entities (#219)** — Installs that existed before v2.14.3 accumulated two orphaned `binary_sensor` registry rows (`Sun Infront` and `Manual Override`) that showed as permanently unavailable. A one-shot migration now removes them automatically on the next restart — no manual cleanup required.

### 🧪 Testing
- Added 7 tests covering the migration: orphan removal, modern entities left intact, idempotency, flag gating, and domain safety
