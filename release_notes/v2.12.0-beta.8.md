# v2.12.0-beta.10 — Winter Insulation Mode

## 🎯 Highlights

Adds winter insulation mode to close non-sun-exposed covers during cold weather, keeping heat inside and reducing heat loss through windows.

## ✨ Features

### Winter Insulation Mode (Issue #29)

- **Auto-close non-sun covers**: In winter mode, closes covers that aren't exposed to direct sun
- **Heat retention**: Reduces heat loss through windows not collecting solar gain
- **Smart seasonal behavior**: Opens sun-exposed covers for passive heating
- **Climate-aware**: Uses winter strategy to maximize energy efficiency

### Split Config Screens

- **Dedicated override screens**: Force Override and Motion Override in separate config steps
- **Clearer UX**: Each screen focused on a single override type
- **Easier configuration**: Less scrolling, better organization

### Config Flow Improvements

- **Removed priority numbers**: Cleaner visibility chain in config summary
- **Better organization**: Logical grouping of related settings
- **Reduced clutter**: Less visual noise in the config UI

## 🔧 Configuration

New configuration options enabled by these changes:

- Clearer separation between force override and motion override settings
- Priority chain visible without confusing numeric values
- Enhanced winter mode integration in config flow
