## 📋 Configuration Summary Screen — Beta 4

This beta adds a read-only configuration summary screen to the config flow, making it easy to review all settings before saving.

### ✨ Features

- **Configuration Summary screen** — new step in the initial config flow shown just before creating the entry; displays all configured settings grouped by section (Cover Type, Geometry, Glare Zones, Sun Tracking, Position, Blind Spot, Automation, Manual Override, Motion & Force Overrides, Weather Override, Climate)
- **Summary menu item in Options flow** — accessible at any time from the options menu as "Configuration Summary"; returns to the menu when dismissed
- Only populated sections are shown — empty or default-value fields are omitted to keep the summary concise
- Cover-type-specific geometry fields shown (vertical/awning/tilt)
- Min/max position limits show a "(during sun tracking only)" qualifier when the limit-only-during-tracking toggle is enabled

### 🧪 Testing

- 939 tests passing
- 36 new tests covering `_build_config_summary()` for all sections, conditional visibility, cover-type-specific fields, and a full smoke test

### ⚙️ Technical

- Requires Home Assistant 2024.5.0+
- Tested with Python 3.11 and 3.12
- Summary text injected via `description_placeholders` (same pattern as the existing Sync confirm screen)
- All 13 language files updated with the new step title and menu label
