## 🎯 Duplicate & Sync Covers

### ✨ Features

- **Duplicate existing cover** — When adding a new cover integration, a new "Duplicate existing cover" option appears. Select a source cover, enter only the new name, cover entities, and azimuth — all other settings (window dimensions, sill height, automation, climate, motion control, etc.) are copied automatically. Perfect for homes with many identical windows.
- **Sync settings to other covers** — Every cover's options menu now includes a "Sync settings to other covers" option. Select one or more covers of the same type and push all shared settings to them in one click. Each cover keeps its own name, entities, azimuth, and device association.

### 🔧 Technical Details

- Shared settings helper `_extract_shared_options` excludes per-window fields (entities, azimuth, device ID) and copies everything else
- Name conflicts during duplicate are resolved automatically with a `(Copy)` / `(Copy 2)` suffix
- Sync only lists covers of the same type (vertical → vertical, etc.)
