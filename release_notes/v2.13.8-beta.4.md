# v2.13.8-beta.4 — Config Summary & UI Improvements

## 🎯 Summary

Improves the configuration summary display by showing sunrise and sunset timing details, and reorganizes the Position Settings screen for better UX by moving the Open/Close threshold control to its logical home.

## ✨ Features

- **Configuration summary now shows after-sunrise position transitions**:
  - Displays when automatic tracking resumes after sunrise (+ offset if non-zero)
  - Shows sunset position transition timing when configured
  - Annotates both sunset and sunrise offsets: e.g., "After sunset (+30 min) → 80%"
  - Helps users understand the full day/night position cycle at a glance

## 🔧 UI/UX Changes

- **Position Settings screen reorganization**:
  - Moved "Open/Close threshold" from Schedule & Timing screen to Position Settings screen (below Sunrise Offset)
  - Rationale: This setting controls position behavior (when calculated position triggers open vs. close), not scheduling
  - Makes the Position Settings screen the single place to configure all position-related thresholds and behaviors
