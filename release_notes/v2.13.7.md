# Release v2.13.7

## 🐛 Bug Fixes

- **Motion Control:** Fixed motion timeout pending state ignoring new motion events (#119)
  - When motion status showed "Timeout Pending" (countdown running, not yet expired), new motion sensor events were being silently ignored
  - The Motion Status sensor would get stuck in "timeout_pending" and covers would not resume automatic sun positioning
  - Now properly detects and responds to motion events during the timeout countdown, immediately cancelling the timeout and resuming automatic control

## 🧪 Testing

- Added 6 new tests covering motion timeout pending state transitions and return value behavior
- All 1099 tests passing
- Tested with Python 3.11 and 3.12
- Home Assistant 2024.5.0+
